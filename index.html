Usage

  • 1. Select the array size using the appropriate slider.
  • 2. Select the sorting speed using the appropriate slider.
  • 3. Click on the desired sorting method and watch sorting take place in real time.
  • Sorting can be stopped at any time by pressing the esc key. New array will need to be generated before any subsequent sorting.

Color coding

  • - bar idle (not being used)
  • - bar selected
  • - bar being swapped
  • - bar being observed
  • - bar in its final position
